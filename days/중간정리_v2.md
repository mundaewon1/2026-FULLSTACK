## 개발 난이도 (회원,관리자,제휴업체관리,행사관리,제휴업체관리) > (카테고리,공지사항,지역관리)

-- 보라님
- 회원(비밀번호 초기화 시 추후 해당 사용자 메일로 임시 비밀번호 발송[외부 API 연동 - 2차])
  - 조회(출석일,회원 상태등으로 조회가능), 회원찾기(회원리스트/페이징)
  - 등록(관리자가 등록 해줌)
  - 수정(상태 - 정지,경고등,기본정보 수정,비밀번호 초기화)
  - 탈퇴(삭제)

-- 욱진님
- 관리자 관리(관리자만 따로관리,가능하면 관리자별 관리자 메뉴 접근 제어[A:지역,공지시항기능사용가능,B:관리자 전체 기능 사용가능])
  - 조회
  - 등록
  - 수정
  - 삭제

-- 일두님
- 지역관리(시도 군구 관리 [2depth])
  - 조회
  - 등록
  - 수정
  - 삭제

-- 대원님
- 카테고리 관리(부모-자식 [2depth])
  - 조회
  - 등록
  - 수정
  - 삭제

-- 혜원님
- 공지사항(제목,내용,게시일,게시상태)
  - 조회
  - 등록
    - 업로드
  - 수정
  - 삭제

-- 예진님
- 행사 관리(이미지,신청서 작성[파일업로드,다운로드],행사일) - 행사 등록 및 참여자 관리
  - 행사 등록
  - 행사 신청목록 확인
  - 행사 신청 승인
  - 수정
  - 삭제

-- 채연님
- 광고 관리(이미지,광고개시 시작,종료일, //사용자 대시보드 게시될 위치선정[상단1,상단2,하단1,하단2등 위치를 정해놓고 해당 위치에 개시되도록]) - 상품광고
  - 조회
  - 등록
  - 수정
  - 삭제




- 제휴 업체 관리(업체 종류[카페,음식점],주소 및 영업시간,네이버 지도 연동[외부 API 연동 - 2차])
  - 조회
  - 등록
  - 수정
  - 삭제
